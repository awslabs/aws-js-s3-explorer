42

This data archive is provided by The Gene Ontology Resource. It contains monthly releases of GO from 2002 to 2018 and was built from the legacy GO CVS, SVN and archive products. To improve reusability, file structure and file format were made consistent with the DOI monthly releases of GO produced since 2018.

Work using the Gene Ontology Resource should cite:

The Gene Ontology Resource: 20 years and still GOing strong.
>Nucleic Acids Res. Jan 2019;47(D1):D330-D338.
Users are also encouraged to cite the release date of GO (e.g. 2005-04-01) used in their analysis to enable the reproducibility of their experiments.
